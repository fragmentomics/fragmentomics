# Contributing

We welcome contributions to FragMentor!

See our full [Contributing Guide](https://github.com/fragmentomics/fragmentomics/blob/main/CONTRIBUTING.md) on GitHub.

## Quick Start

```bash
git clone https://github.com/fragmentomics/fragmentomics
cd fragmentomics
pip install -e ".[dev]"
pytest
```

## Ways to Contribute

- Report bugs
- Suggest features
- Submit pull requests
- Improve documentation
- Share use cases
